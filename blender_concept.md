# Blender 3D Concept (Optional Upgrade)

If you want a full cinematic 3D version:

1. **Model setup**
   - Create two glass reactors/flasks and a lab bench in Blender.
   - Use emissive particle instances for molecule types (N2, H2, NH3).
2. **Animation blocks**
   - Block A: Dynamic equilibrium loops with particle flow in both directions.
   - Block B: Pressure change using container scale keyframes.
   - Block C: Temperature disturbance using heat glow and volumetric light.
3. **Data-driven overlays**
   - Export transparent PNGs or EXR sequences.
   - Composite with Manim text overlays for formulas, labels, and exam sentences.
4. **Camera language**
   - Slow dolly-in for concept introductions.
   - Side-by-side split view for disturbance comparison.
5. **Render**
   - Eevee for speed, Cycles for final quality.
   - 24 fps, 1920x1080, color-managed filmic look.

This repository keeps the fully working lesson in Manim first, and this file documents the Blender path without adding heavy assets.
